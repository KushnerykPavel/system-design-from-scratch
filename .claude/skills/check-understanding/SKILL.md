# Check Understanding

Quiz the learner on one phase using `quiz.json` files from the phase lessons.

Prioritize questions that probe:
- trade-offs
- failure modes
- observability
- sizing impact

## Verdicts

- `8/8`: mastered
- `6-7`: solid
- `4-5`: shaky
- `0-3`: revisit
