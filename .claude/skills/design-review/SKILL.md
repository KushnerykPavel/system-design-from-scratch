# Design Review

Activation:
- `/design-review <phase> <lesson>`

Review dimensions:
- clarification quality
- sizing completeness
- architecture coherence
- failure-mode depth
- observability
- trade-off clarity

Output:
- strengths
- gaps
- one highest-leverage improvement
