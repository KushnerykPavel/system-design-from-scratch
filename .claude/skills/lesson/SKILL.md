# Lesson

Activation:
- `/lesson <phase> <lesson>`

Workflow:
1. Read the lesson doc.
2. Ask the learner to clarify the prompt.
3. Ask for rough sizing.
4. Ask for the high-level architecture.
5. Push on one deep dive.
6. Push on failure modes and observability.
7. Run the post-lesson quiz.
