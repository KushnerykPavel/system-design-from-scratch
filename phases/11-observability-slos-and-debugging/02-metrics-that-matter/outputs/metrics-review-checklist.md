# Metrics Review Checklist

- Which metric best represents user pain?
- Which metric best reveals throughput or demand shape?
- Which metric best reveals saturation or capacity pressure?
- Which metric best reveals dependency trouble?
- If async work exists, what metric reveals delay or staleness?
- Which labels are bounded and safe?
- Which proposed labels are likely to explode series count?
- Can an on-call engineer explain the first next check after seeing each panel?
