# Skill — Consumer Backend Mock

## 20-Minute Solo Drill

- Minute 0-3: clarify one user journey and one unacceptable failure
- Minute 3-6: size traffic, skew, and retention
- Minute 6-11: draw high-level architecture
- Minute 11-15: deep dive on one bottleneck
- Minute 15-18: failure modes and observability
- Minute 18-20: redesign after a changed constraint

## Self-Check

- Did I say what matters most: freshness, delivery confidence, ranking quality, or cost?
- Did I estimate one amplification factor?
- Did I name a source of truth?
- Did I explain what users see during a partial outage?
- Did I change the design meaningfully during redesign?
