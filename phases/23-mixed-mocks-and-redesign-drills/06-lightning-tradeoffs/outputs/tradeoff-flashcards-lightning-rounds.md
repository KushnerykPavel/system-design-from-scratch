# Trade-off Flashcards — Lightning Rounds

## Prompts

- read-through cache vs write-through cache
- active-active vs active-passive
- push fanout vs pull fanout
- strong consistency vs bounded staleness
- fail-open vs fail-closed
- direct sync call vs async queue
- shared multi-tenant cluster vs stronger isolation tier

## For Each Card, Answer

- what requirement wins?
- which option do I choose?
- what gets worse?
- what metric proves I was wrong?
