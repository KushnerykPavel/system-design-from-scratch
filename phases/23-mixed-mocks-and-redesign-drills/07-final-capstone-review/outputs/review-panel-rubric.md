# Review Panel Rubric

Score each dimension from 1 to 4.

| Dimension | What a 4 looks like |
|-----------|---------------------|
| clarification | bounds the problem quickly and cleanly |
| sizing | estimates the right numbers and uses them |
| architecture | presents a coherent high-level design |
| deep dive | chooses the riskiest area and explains it well |
| failure modes | names realistic failures and containment |
| observability | connects signals to user or operator outcomes |
| trade-offs | states what was chosen and what was sacrificed |
| communication | stays calm, structured, and revisable under pressure |
