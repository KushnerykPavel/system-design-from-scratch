# Phase 00 — Setup & Workflow

~8 hours.

This phase builds the operating system for the rest of the course. The goal is not to memorize infrastructure yet. The goal is to install the habits that make every later design answer sharper: structured notes, fast diagrams, reusable capacity math, honest debriefs, and a repeatable mock workflow.

## Outcomes

By the end of phase `00`, you should be able to:

- set up the repo and progress tracking without friction
- capture architecture notes in a reusable format
- draw a high-level system diagram quickly under time pressure
- maintain a mistake log that turns weak answers into better next attempts
- use lightweight capacity sheets before architecture detail explodes
- review answers with a senior-level checklist
- run mock interviews with a consistent flow
- debrief and schedule the next iteration deliberately
