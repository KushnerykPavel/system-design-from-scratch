## Timed drill flow

1. Clarify top reads, writes, invariants, and retention needs.
2. Rank access patterns.
3. Choose one primary store and key layout.
4. Add only necessary indexes or derived systems.
5. Close with hot/cold strategy, deletion semantics, and observability.

## Deliverables
- One system of record
- One primary key
- One secondary query path
- One lifecycle or deletion workflow
- One trade-off summary
