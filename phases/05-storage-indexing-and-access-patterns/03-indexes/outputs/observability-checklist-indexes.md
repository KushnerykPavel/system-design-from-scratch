## Metrics
- Write latency by table and number of maintained indexes
- Index storage growth rate
- Query volume by indexed versus fallback path
- Backfill progress and lag

## Logs
- Rejected unbounded query shapes
- Planner fallback to scan or degraded path

## Alerts
- Commit latency regression after index rollout
- Index build backlog exceeds planned window
- Secondary index freshness or repair lag breaches threshold
