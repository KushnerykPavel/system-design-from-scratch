# Interview Card — Secure Defaults Drill

## Strong answer shape

- Make the paved path safe.
- Deny broadly, grant explicitly.
- Use managed identity and secret delivery.
- Add default quotas, deletion policy, and auditable overrides.
- Explain degraded mode.

## Common misses

- Security added only as optional follow-up.
- Broad permanent overrides.
- No tenant-scoped defaults.
- No deletion or retention stance.
