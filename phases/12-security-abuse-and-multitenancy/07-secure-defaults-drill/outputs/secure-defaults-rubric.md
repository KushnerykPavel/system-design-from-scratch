# Secure Defaults Rubric

Score each answer on whether it includes:

- deny-by-default permissions
- managed short-lived credentials
- default quotas or abuse controls
- tenant-scoped storage and async context
- deletion and retention defaults
- expiring overrides
- degraded-mode behavior

## Interpretation

- `12-14`: strong, senior-level secure-defaults answer
- `8-10`: developing, good instincts but incomplete
- `0-6`: weak, mostly bolt-on security
